Class Hierarchy

  • object: The most base type
    • cmt.ConditionalDistribution: Abstract base class for conditional models.
      • cmt.GLM: An implementation of generalized linear models.
      • cmt.MCBM: An implementation of a mixture of conditional Boltzmann machines.
      • cmt.MCGSM: An implementation of a mixture of conditional Gaussian scale mixtures.
    • cmt.Distribution: Abstract base class for distributions.
      • cmt.PatchModel: Abstract base class for models of image patches.
        • cmt.FVBN: Model image patches using a GLM for each conditional distribution.
        • cmt.PatchMCBM: Model image patches by using an MCBM for each conditional distribution.
      • cmt.UnivariateDistribution: Abstract base class for univariate distributions usable by GLM.
        • cmt.Bernoulli: The Bernoulli distribution.
    • cmt.Nonlinearity: Abstract base class for nonlinear functions usable by GLM.
      • cmt.LogisticFunction: The sigmoidal logistic function.
    • cmt.Preconditioner: Abstract base class for preconditioners of inputs and outputs.
      • cmt.AffinePreconditioner: Performs affine transformations on the input and the output.
        • cmt.AffineTransform: Performs an affine transformation on inputs.
          • cmt.PCATransform: This class behaves like PCAPreconditioner, but does not change the output.
          • cmt.WhiteningTransform: Decorrelates inputs.
        • cmt.PCAPreconditioner: This preconditioner can be used to reduce the dimensionality of the input.
        • cmt.WhiteningPreconditioner: Decorrelates inputs and outputs.
