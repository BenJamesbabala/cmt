Module cmt

This module provides C++ implementations of conditional models such as the MCGSM, as well as tools for solving applications such as filling-in, denoising, or classification.

fill_in_image(img, model, xmask, ymask, fmask, preconditioner=None, num_iter=10, num_steps=100)

 

Samples pixels of an image conditioned on all other pixels.

Parameters:
  • img (ndarray) - the image with the missing pixels initialized somehow
  • model (ConditionalDistribution) - a conditional distribution such as an MCGSM
  • xmask (ndarray) - a Boolean array describing the input pixels
  • ymask (ndarray) - a Boolean array describing the output pixels
  • fmask (ndarray) - a Boolean array describing the missing pixels
  • preconditioner (Preconditioner) - transforms the input before feeding it into the model
  • num_iter (int) - number of iterations of replacing all pixels
  • num_steps (int) - number of Metropolis steps per pixel and iteration
Returns: ndarray
an image with the missing pixels replaced

sample_image(img, model, xmask, ymask, preconditioner=None)

 

Generates an image using a conditional distribution. The initial image passed to this function is used to initialize the boundaries and is also used to determine the size and length of the image to be generated.

Parameters:
  • img (ndarray) - initialization of image
  • model (ConditionalDistribution) - a conditional distribution such as an MCGSM
  • xmask (ndarray) - a Boolean array describing the input pixels
  • ymask (ndarray) - a Boolean array describing the output pixels
  • preconditioner (Preconditioner) - transforms the input before feeding it into the model
Returns: ndarray
the sampled image
