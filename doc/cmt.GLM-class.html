Class GLM

                         GLM

An implementation of generalized linear models.

$$p(y \mid \mathbf{x}) = q(y \mid g(\mathbf{w}^\top \mathbf{x} + b)),$$

where $q$ is typically from the exponential family and $g$ is some nonlinearity (inverse link function) which has to be specified.

To perform logistic regression, for example, define a GLM with LogisticFunction and Bernoulli distribution,

>>> glm = GLM(inputs.shape[0], LogisticFunction, Bernoulli)
>>> glm.train(inputs, outputs)

To access the linear filter $\mathbf{w}$ or the bias $b$ use

>>> glm.weights
>>> glm.bias
